# About The Clinic Management System
The Clinic Management System is a C# console-based application developed during my second year of Software Engineering.

The purpose of this project was to apply object-oriented programming principles and build a structured, menu-driven system capable of managing patient records in a simulated clinic environment.

The system allows users to add, remove, search, and display patient information. It also includes functionality to export patient data to a text file using file handling techniques.

This project demonstrates foundational software development concepts such as class design, collections, LINQ queries, file I/O operations, and exception handling in C#.

# Features
- Add new patients
- Remove existing patients
- Search patients by name
- Display all registered patients
- Export patient information to a text file

# Technologies Used
- C#
- .NET Console Application
- Object-Oriented Programming (OOP)
- LINQ
- File Handling
